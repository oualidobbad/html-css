    # html-css

    Practice playground for layouts, typography, and components with plain HTML/CSS.

    ## Contents
    - Layout experiments (flex/grid), buttons, forms, and small UI blocks.
    - Theming trials with variables and utility classes.

    ## Usage
    - Open any HTML file in a browser; no build step.
    - Tweak CSS and refresh to iterate quickly.

    ## Notes
    - Keep assets relative for easy offline preview; test in modern evergreen browsers.
